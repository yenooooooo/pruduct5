# HabitFlow

하루 하나, 인생을 바꾸는 습관을 만들어보세요.

## 주요 기능

*   **하루 딱 하나의 습관:** 가장 중요한 습관에 집중하여 성공 확률을 높입니다.
*   **간편한 기록:** 버튼 클릭 한 번으로 오늘의 습관을 기록할 수 있습니다.
*   **연속 기록 시각화:** 연속 성공 기록을 통해 성취감을 느끼고 동기를 부여받으세요.
*   **PRO 기능:**
    *   습관 무제한 생성
    *   연속 기록 통계 확인
    *   히스토리 관리

## 기술 스택

*   HTML
*   CSS
*   JavaScript
*   Firebase (Authentication, Firestore)

## 시작하기

1.  이 저장소를 클론하세요.
2.  `index.html` 파일을 브라우저에서 여세요.
3.  회원가입 또는 로그인하여 습관 관리를 시작하세요.
